# 🚕 Taxi Driver

A fast-paced urban racing game built in Unity — pick up passengers, drop them off, and choose your ride.

---

## 🎬 Trailer

[![Taxi Driver Trailer](https://img.youtube.com/vi/nIZjB03Dad0/maxresdefault.jpg)](https://www.youtube.com/watch?v=nIZjB03Dad0)

---

## 🎮 Gameplay

- **Pick up passengers** scattered across a living city environment
- **Race to destinations** as fast as possible to maximize your score
- **Select your car** from a roster of driveable vehicles before each run
- Avoid traffic, nail your routes, and keep the fares coming

---

## 🛠️ Built With

- **Engine:** Unity 2022.3 LTS
- **Render Pipeline:** URP (Universal Render Pipeline)
- **Car Physics:** Realistic Car Controller V4
- **City Assets:** PolygonCity (Synty Studios)

## 🚀 Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/sac1441/Taxi-Driver.git
   ```
2. Open the project in **Unity 2022.3 LTS**
3. Open `Assets/PolygonCity/Scenes/Final.unity`
4. Press **Play**

> ⚠️ Requires Unity 2022.3.x — other versions may have compatibility issues with RCC V4.
